# Personal Finance Tracker

   Aplikasi untuk mengelola pengeluaran harian menggunakan PHP dan MySQL. Dibangun dengan fokus pada keamanan data dan visualisasi laporan.

   ## Fitur Utama:
   * 📊 Visualisasi Dashboard (Chart.js)
   * 💸 Manajemen Pemasukan & Pengeluaran
   * 📑 Laporan Riwayat Transaksi
   * 🔒 Keamanan Database dengan PDO

   ## Tech Stack:
   * **Language:** PHP
   * **Database:** MySQL
   * **Styling:** Tailwind CSS / Bootstrap
   * **Environment:** Ubuntu Linux
